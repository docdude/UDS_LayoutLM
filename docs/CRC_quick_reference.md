## Must-Label Entities
✅ Patient ID  
✅ Date of procedure/test  
✅ Test/procedure type  
✅ Result  
✅ CPT & ICD-10 codes  

## Common CPT Codes
- 45378 = Diagnostic colonoscopy
- 45385 = Colonoscopy + polypectomy
- 82274 = FIT test
- 82270 = FOBT

## Common ICD-10 Codes
- Z12.11 = CRC screening
- K63.5 = Colon polyp
- Z86.010 = History of polyps
